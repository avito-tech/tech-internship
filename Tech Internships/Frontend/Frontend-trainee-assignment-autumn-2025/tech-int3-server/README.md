# API сервер для модерации объявлений

Express.js сервер для задания по модерации объявлений.

## Описание

Упрощенная версия системы модерации объявлений платформы Авито. Сервер предоставляет REST API для управления объявлениями, просмотра статистики и работы модераторов.

## Технические требования

- Node.js v20+
- Express.js

## Установка и запуск

```bash
# Переход в директорию сервера
cd tech-int3-server

# Установка зависимостей
npm install

# Запуск сервера
npm start

# Или запуск в режиме разработки с автоматической перезагрузкой
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:3001`

## Документация API

API задокументировано в файле `schema.yaml` в формате OpenAPI 3.0. Файл содержит полное описание всех endpoints, схем данных и примеров использования.

### Версионирование API
API использует версионирование через URL path: `/api/v1/`
